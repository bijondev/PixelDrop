## What Is PixelDrop

PixelDrop is a desktop image utility that helps you compress, resize, and organize images faster from one lightweight app.

It is useful for:

- designers
- developers
- content creators
- marketers
- ecommerce teams
- anyone who works with many images every day

## Benefits

- Saves time by handling image tasks from one place
- Reduces image file size for faster uploads and smaller storage usage
- Makes bulk image processing easier with batch tools
- Helps keep image output consistent with presets and folder rules
- Supports automatic processing through watched folders
- Keeps a local history of processed files
- Works as a desktop-first tool without needing a browser workflow

## Main Features

### Watched Folders

- Watch selected folders automatically
- Compress new images as they appear
- Set per-folder format, quality, suffix, and output location

### Resize Tools

- Single image resize
- Batch resize
- Preset sizes for common use cases
- Custom width and height support
- Retina `@2x` output option

### Compression Tools

- Single image compression
- Multi-file compression
- Folder-based compression input
- Format selection: original, JPG, PNG, WebP, AVIF
- Quality control
- Custom output suffix

### History

- Stores processed file records locally
- Shows recent activity
- Helps track output and basic savings
- Supports undo when original backups exist

### Settings

- Default quality
- Default format
- Default fit
- Theme switching
- Launch at startup

## Why Use PixelDrop

Use PixelDrop if you want:

- a simple desktop app instead of a complicated image pipeline
- faster image preparation before upload or publishing
- automatic compression for folders where new images are added often
- consistent image sizes for social media, websites, or product catalogs
- a lightweight local workflow with no need to depend on online tools

## Good Use Cases

- preparing website assets
- compressing product images for ecommerce
- creating social media image sizes
- cleaning up screenshots before sharing
- reducing image sizes before email or upload
- processing export folders from design tools

## Installer

Run:

```text
PixelDrop-Setup.exe
```

After installation, PixelDrop can be added to the Windows Start Menu / Programs list for easier access.
